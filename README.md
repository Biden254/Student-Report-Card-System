## Project Overview

This project is a Java console application that collects full student performance data and generates a formatted scoresheet similar to the sample provided in the assignment instructions.

The program demonstrates key programming concepts including:

User input with Scanner

Arrays for storing student records

Loops for processing data

Control statements for ranking logic

Formatted output using printf

Computing totals, averages, and grade distribution

The final output replicates the exact alignment and structure of the sample scoresheet.
